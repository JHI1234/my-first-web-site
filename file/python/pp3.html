📝 개념 정리

  1. 변수(Variable)
    2. 개념 : 데이터를 저장하기 위해 컴퓨터 안에 만들어 놓은 메모리 공간
    - 변수에 새로운 데이터를 대입하면 이전에 저장된 데이터는 사라지고 새로운 데이터가 저장됨.
  2. 자료형(Data type)
    3. 개념 : 데이터의 형식
    종류 : 문자열, 숫자(정수,실수), 리스트, 튜플, 딕셔너리, 집합 등
    - 파이썬에서는 데이터의 자료형을 쉽게 알 수 있도록 돕는 type()이라는 함수를 제공함.
  3. 숫자형
    4. 종류 : 정수형(Integer type)과 실수형(Float type)이 파이썬에서 대표적인 숫자 자료형임.
    - int는 정수형을, float은 실수형을 나타냄.
  4. 불 자료형 : 참(True)과 거짓(False)을 값으로 가지는 자료형
    5. 주의사항 : True와 False의 대소문자를 정확하게 작성하지 않을 시 오류가 발생함.
  5. 변수 이름 규칙
    • 변수 이름은 숫자로 시작할 수 없다.
    • 변수 이름에는 공백을 사용할 수 없다.
    • 예약어는 변수 이름으로 사용할 수 없다.
    • 변수 이름은 스네이크 케이스 규칙에 따라 짓는다.
    • 변수 이름은 저장할 데이터의 의미를 담아 짓는다.
