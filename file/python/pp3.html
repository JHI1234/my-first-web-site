📝 개념 정리

  1. 변수(Virable)
    2. 개념 : 데이터를 저장하기 위해 컴퓨터 안에 만들어 놓은 메모리 공간
    - 변수에 새로운 데이터를 대입하면 이전에 저장된 데이터는 사라지고 새로운 데이터가 저장됨.
  2. 자료형(Data type)
    3. 개념 : 데이터의 형식
    종류 : 문자열, 숫자(정수,실수), 리스트, 튜플, 딕셔너리, 집합 등
    - 파이썬에서는 데이터의 자료형을 쉽게 알 수 있도록 돕는 type()이라는 함수를 제공함.
  3. 숫자형
    4. 종류 : 정수형(Integer type)과 실수형(Float type)이 파이썬에서 대표적인 숫자 자료형임. - int는 정수형을, float은 실수형을 나타냄.
